**Course Description**:    
This course provides the students with an understanding in object-oriented programming with emphasis on developing large-scale projects. These topics will be further explained in lab sessions using Java as the programming language. More advanced programming concepts such as concurrency, generics and persistence will also be taught. After successfully completing this course, students will have a good understanding of the object oriented analysis and design process, and its implementation in Java.
